# --- Modified Cleaning Step (corrected) ---

# Drop columns with too many missing values or not useful for modeling
# errors='ignore' prevents KeyError if column is missing
gdsc_ic50 = gdsc_ic50.drop(columns=['TCGA_DESC'], errors='ignore')  

# Keep only rows where PUTATIVE_TARGET is present (helps for drug-specific modeling)
gdsc_ic50 = gdsc_ic50[gdsc_ic50['PUTATIVE_TARGET'].notna()]  

# Fill missing values in Cell Lines dataset
cell_lines = cell_lines.fillna('Unknown')

# Convert Yes/No columns to binary for ML
binary_cols = ['Whole Exome Sequencing (WES)', 'Copy Number Alterations (CNA)',
               'Gene Expression', 'Methylation', 'Drug\nResponse']
for col in binary_cols:
    cell_lines[col] = cell_lines[col].map({'Y':1, 'N':0, 'Unknown':0})

# Rename columns for easier merging later
cell_lines = cell_lines.rename(columns={
    'Sample Name':'CELL_LINE_NAME',
    'COSMIC identifier':'COSMIC_ID',
    'Cancer Type\n(matching TCGA label)':'Cancer_Type',
    'Microsatellite \ninstability Status (MSI)':'MSI_Status'
})

Data Cleaning Summary: I removed the TCGA_DESC column from the GDSC2 dataset due to its high number of missing and non-informative text values. Rows with missing PUTATIVE_TARGET values were also dropped to keep the data consistent. In the Cell Lines dataset, missing categorical values (like cancer type and tissue descriptors) were filled with 'Unknown' to avoid data loss. Column names were standardized and null values checked to ensure data quality. These steps improved dataset consistency and reduced noise for better model performance.

# Create binary labels (1=sensitive, 0=resistant)
